Display SSL Seal or Custom Image on WooCommerce Checkout Page.

== Description ==

WooCommerce is the most popular plugin in use to power WordPress store fronts and now you can show your customers that your site is secure while they checkout. It is a must that if you are accepting credit cards on your site, you need to have a SSL Certificate installed and setup. Take this a step further and let your customers know that the page they are entering their credit card information on is 100% secure.

This plugin will let you:

 - Display a default image
 - Display custom image
 - Embed a Secure Seal SSL provider.

All displays will be directly under the PLACE ORDER button on your WooCommerce checkout page.

= Where to Find Settings Screen =
  
  WooCommerce > Settings > SSL Seal Tab

<strong>There is NO BETTER WAY to show your customers how secure your WooCommerce site is.</strong>


== Installation ==

= Install from WP Dashboard =
  * Log into WP dashboard then click **Plugins** > **Add new** > Then under the title "Install Plugins" click **Upload** > **choose the zip** > **Activate the plugin!**

= Install from FTP =
  * Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WP installation and then activate the Plugin from Plugins page.

= THAT IS IT: You're done! =
